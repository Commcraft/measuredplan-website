Every project starts from the same principle: the drawings should describe the
building precisely, so that everyone working from them - designer, architect,
contractor - is looking at the same accurate record.

We measure on site, then draft in CAD to a consistent layer structure and a
clear title block, so the set reads cleanly at the scale it is issued for.
